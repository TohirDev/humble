## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/akumarujon/website.git
   ```
2. Navigate into the project directory:
   ```bash
   cd website
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage

Run the project in the command line.

```bash
npm start
```
